MeshData_

type, public :: MeshData_

MeshData_ contains data related to a mesh

Note

In EASIFEM mesh-data is separated from the Mesh_ mainly because very often we only require connectivity information which is already stored inside the Mesh_.

MeshData_ is contains mesh realated data
