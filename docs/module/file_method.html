  • private subroutine write_data_ascii_r1(Obj, Val, row, col)

    This routine writes data into a file If row is present then data is printed as row If col is present then data is printed as column If both row or col are absent then data is printed as row

    Arguments

    Type IntentOptional AttributesName
    class(File_), intent(inout) :: Obj

    File object
    class(*), intent(in) :: Val(:)

    One D array
    logical(kind=LGT), intent(in), optional :: row

    If present then vector will be printed as rowwise
    logical(kind=LGT), intent(in), optional :: col

    If present then vector will be printed as column wise
