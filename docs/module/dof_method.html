interface

Set values in a vector of real numbers

  • private subroutine dof_addValue_1(Vec, Obj, Nptrs, Val, scale, Conversion)

    This subroutine is designed to add values in a vector of real number - DOF_ object obj contains the storage pattern of degrees of freedom inside Vec. This storage pattern can be FMT_Nodes or FMT_DOF - Val denotes the nodal values of all dof defined inside Obj. Once storage pattern in Val can be FMT_DOF or FMT_Nodes. - To tackle this Conversion can be set to DOFToNodes, NodesToDOF or NONE.
