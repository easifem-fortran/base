Nodes of different colours represent the following:

Solid arrows point from a derived type to the parent type which it extends. Dashed arrows point from a derived type to the other types it contains as a components, with a label listing the name(s) of said component(s). Where possible, edges connecting nodes are given different colours to make them easier to distinguish in large graphs.
