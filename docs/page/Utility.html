Utility

Utility module in EASIFEM contains useful routines for doing small but important tasks.

@StringMethods

call display( "test: StringMethods" )

call is( UpperCase( "hello small world" ), "HELLO SMALL WORLD", "test:1" )

call is( UpperCase( "ab CeD89qu3rhkjBGK" ),"AB CED89QU3RHKJBGK", "test:2")

call is( LowerCase( "HELLO worLD" ),"hello world", "test:3")

call is( isWhiteChar( ' ' ), .TRUE., 'test:4' )

call is( isWhiteChar( ' ' ), .TRUE., 'test:5' )

call is( isWhiteChar( '1' ), .FALSE., 'test:6' )

call is( isWhiteChar( '' ), .FALSE., 'test:7' )

call is( isBlank( ' ' ), .TRUE., 'test:8' )

call is( isBlank( ' 1 ' ), .FALSE., 'test:9' )

call is( numStrings( ' 1 ' ), 1, 'test:10' )

call is( numStrings( ' ' ), 0, 'test:11' )

call is( numStrings( '1 2 3 4 5 6' ), 6, 'test:12' )

call is( numStrings( '1 3 45 sgfsd "hsk ssdsdf " 3 4 5 6' ), 9, 'test:13')

call is( numMatchStr( "hello world, he or she, hesitant", "he" ), &

& 4, 'test:13' )

ToUpperCase

This is a subroutine which converts a string into upper case.

UpperCase

This is a function which returns a string in upper case.

ToLowerCase

This is subroutine which converts a string into upper case

LowerCase

This is a function which converts a string into lower case

isWhiteSpace

This function returns true when the given char is a space or a tab

isBlank

This function returns true when given string is completely blank.

numStrings

This function returns the number of substrings inside a string which are separate by one or more spaces.

numMatchStr

This function returns the number of pattern present inside the string.

strFind

This routine finds the pattern inside the string, and returns the indices.

FindReplace

This routine finds the pattern inside the string, and replace it with the given pattern.

getField

This routine returns the field from the string.

getFileParts()

getFileName()
